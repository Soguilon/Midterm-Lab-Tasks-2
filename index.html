üìù Step-by-Step Process

    Step 1:

    • Download the dataset (Uncleaned_DS_jobs.csv)
    • Open Excel
    • Go to Data ‚Üí New Query ‚Üí Open File ‚Üí Text/CSV
    • Click Load and then Edit using Power Query Editor

    Step 2:

    • Right-click the dataset in the Queries pane
    • Select Duplicate

    Step 3: Clean Salary Data

    • Select the Salary Estimate column
    • Go to Transform Menu ‚Üí Extract ‚Üí Text Before Delimiter
    • Type "(" and click OK
    • Create two new columns: Min Salary and Max Salary
    • Rename the columns to "Min Sal" and "Max Sal"

    Step 4: Add Role Type Column

    • Go to Add Column Menu ‚Üí Custom Column
    • Use logic to assign roles based on job titles

    Step 5: Split Location Column

    • Split the Location column by custom rules (e.g., assigning state abbreviations)

    Step 6: Split Size Column

    • Create MinCompanySize and MaxCompanySize columns using the same method as Salary Estimate

    Step 7: Handle Negative Values

    • Filter out invalid values like -1 and 0 from relevant columns

    Step 8: Clean Company Names

    • Remove extra characters or ratings from company names

    Step 9: Copy Cleaning Steps as Proof

    • Go to Home Menu ‚Üí Click Advanced Editor
    • Copy and save the code in your portfolio

    Step 10: Reshape and Group Data

    • Group by Role Type and Company Size
    • Get average Min and Max Salary

    Step 11: Merge State Mapping

    • Merge data from State Mapping query to get the full state name

    Step 12: Group by State

    • Group by state and get the average Min and Max Salary

    Step 13: View Query Dependencies

    • Check if all queries are correctly linked using View Menu ‚Üí Dependencies
